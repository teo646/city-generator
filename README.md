# cityFilter
generative art filter that draw city based on given image

### Prerequisites

cv2, numpy but will be install automatically via pip install -e.

### Installing

1. create virtual environment(venv)

2. install module
```
cd maskCanvas
pip install -e .
```


## Running the tests
```
python example/masking.py
```
